# 多通道波形CSV生成工具

基于 Electron 的桌面端波形数据生成工具，通过可视化界面配置参数，生成多通道标准波形数据并导出 CSV 文件。

> 🤖 **开发方式**：Vibecoding 需求驱动开发，由 EasyClaw + DeepSeek V4 Pro 模型完成全部编码与调试。

## 功能特性

### 波形类型
- **正弦波**：可配置幅值、频率、直流偏移量、初始相位
- **方波**：可配置幅值、频率、占空比
- **三角波**：可配置幅值、频率
- **均匀随机数**：可配置数值上限、下限

### 全局参数
- 时间间隔（ms）、数据长度（采样点数）、通道数（1~10 路）

### 波形预览
- Canvas 实时波形预览，参数变更后自动刷新
- **鼠标滚轮缩放**：以鼠标位置为中心缩放 X 轴
- **Shift+拖拽框选**：框选区域后放大
- **拖拽平移**：直接拖拽画布移动视图
- **工具栏**：放大/缩小/适应窗口/重置按钮
- **通道显隐**：点击图例切换各通道波形显示

### CSV 输出
- 根目录：`D:\data\`
- 子文件夹：`YYYYMMDD_HHMMSS`（当前系统时间）
- 文件名：`YYYYMMDD_HHMMSS.csv`
- 编码：UTF-8 BOM
- 格式：保留 2 位小数，无科学计数法
- 表头：`时间(ms),V1,V2,...,Vn`

### 预设管理
- 保存/加载配置方案，本地 JSON 持久化，重启不丢失

## 技术栈

| 模块 | 技术 |
|------|------|
| 桌面框架 | Electron 28 |
| 前端 | 原生 HTML + CSS + JavaScript |
| 波形绘图 | Canvas |
| 文件读写 | Electron 内置 fs API |
| 预设存储 | 本地 JSON 文件 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

## 打包

```bash
# Windows 安装包
npm run dist
```

## 系统要求

- 需具备 `D:\data\` 目录写入权限
