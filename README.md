# 多通道波形CSV生成工具 V1.0

## 项目简介
轻量化单页面桌面应用，通过可视化界面配置参数，生成多通道标准波形数据，并导出规范的 CSV 文件，可用于设备测试、仿真调试、数据校验等场景。

## 技术栈
- Electron 28.0.0
- 原生 HTML + CSS + JavaScript
- Canvas 波形预览

## 快速启动

### 安装依赖
```bash
npm install
```

### 运行
```bash
npm start
```

### 打包
```bash
npm run dist
```

## 功能说明

### 全局参数
- **时间间隔**：相邻采样点的时间步长（ms），0.1~10000
- **数据长度**：采样点数，即 CSV 有效数据行数
- **通道数**：1~10 路，列名固定为 V1 ~ Vn

### 波形类型
1. **正弦波**：可配置幅值、频率、直流偏移量、初始相位
2. **方波**：内置固定参数（幅值=5, 频率=1kHz, 占空比=50%）
3. **三角波**：内置固定参数（幅值=5, 频率=1kHz）
4. **均匀随机数**：可配置数值上限、下限

### CSV 输出
- 根目录：`D:\data\`
- 子文件夹命名：`YYYYMMDD_HHMMSS`
- 文件格式：UTF-8 编码，保留 2 位小数
- 表头格式：`时间(ms),V1,V2,...Vn`
