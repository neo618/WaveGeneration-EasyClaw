# 多通道波形CSV生成工具 需求文档
**适配场景**：Vibecoding 需求驱动开发
**开发框架**：Electron
**文档版本**：V1.0
**目标平台**：Windows 桌面端
**运行模式**：单机离线运行（无需联网、无后端依赖）
**文档状态**：需求最终确认版

---

## 1. 项目概述
### 1.1 项目背景
开发一款桌面端波形数据生成工具，通过可视化界面配置参数，生成多通道标准波形数据，并导出为规范的 CSV 文件，可用于设备测试、仿真调试、数据校验等场景。

### 1.2 产品定位
轻量化单页面桌面应用，功能聚焦**参数配置、波形预览、CSV 导出**，无冗余拓展功能，操作简洁、输出格式统一合规。

### 1.3 设计原则
1. 功能闭环：仅实现已确认需求，不额外新增功能；
2. 格式严格：CSV 输出、文件存储规则强制执行；
3. 稳定优先：异常输入不崩溃、不报错；
4. 离线运行：全程本地计算、本地存储，无网络请求。

## 3. 整体功能总览
1. 全局参数配置：时间间隔、采样点数、通道数；
2. 多通道独立配置：1~10 通道，每通道独立选择波形、配置参数；
3. 波形实时预览：参数变更后自动刷新曲线预览；
4. 参数预设：支持配置方案本地保存、加载；
5. 一键生成 CSV：按固定规则自动创建目录、命名并保存文件；
6. 支持波形类型：正弦波、方波、三角波、均匀随机数。

> 额外约束：**无参数合法性校验、无参数重置按钮、无进度提示、无波形叠加、无批量生成功能**。

## 4. 详细功能需求
### 4.1 全局参数配置模块（全通道共用）
所有参数界面常驻展示，配置项及规则如下：

| 配置项 | 单位 | 输入规则 | 取值范围 | 默认值 | 业务说明 |
| ------ | ---- | -------- | -------- | ------ | -------- |
| 时间间隔 | ms（毫秒） | 仅支持**整数** | 0.1 ~ 10000 ms | 2 ms | 相邻两个采样点的时间步长，决定 CSV 时间列递增规则 |
| 数据长度 | 个 | 正整数 | / | 2048 | 即采样点数，对应 CSV 有效数据行数 |
| 通道数 | 路 | 正整数 | 1 ~ 10 | 2 | 控制物理量列数量，列名固定为 `V1、V2...Vn`，不可自定义 |

### 4.2 多通道波形配置模块
**核心规则**：每个通道配置区域相互独立，波形类型、参数互不影响；选择波形类型后，自动展示/隐藏对应参数输入项。

#### 4.2.1 支持波形类型及配置规则
1. **正弦波**
   - 可配置项：幅值、频率、直流偏移量、初始相位
   - 参数限制：无取值范围，支持小数、负数输入
   - 默认参数：幅值=10，频率=1kHz，直流偏移=0，初始相位=0

2. **方波**
   - 可配置项：**无**
   - 规则：使用程序内置固定参数，用户不可修改

3. **三角波**
   - 可配置项：**无**
   - 规则：使用程序内置固定参数，不支持锯齿波

4. **均匀随机数**
   - 可配置项：数值上限、数值下限
   - 规则：仅生成均匀分布随机数，**不支持随机种子**，数据不可复现；无参数范围限制

### 4.3 波形实时预览模块
1. 载体：使用 Canvas 绘制，固定区域展示在界面右侧；
2. 刷新逻辑：修改任意参数后，预览曲线**自动实时更新**；
3. 展示规则：
   - 多通道曲线使用不同颜色区分，标注对应通道编号；
   - X 轴：时间轴（单位 ms），与当前总采样时长一致；
   - Y 轴：数值轴，自动适配波形幅值范围；
4. 附加约束：仅基础曲线展示，**不支持缩放、拖拽、点位查看**。

### 4.4 参数预设管理模块
1. 保存预设：将「全局参数 + 所有通道波形参数」整体保存为本地方案，支持自定义预设名称；
2. 加载预设：从预设列表选择方案，一键覆盖当前界面所有配置；
3. 数据存储：预设持久化在本地，程序重启后不丢失；
4. 附加约束：仅支持**保存/加载**，无预设删除、导出、导入功能。

### 4.5 CSV 生成与文件存储模块
1. 触发方式：点击界面「生成」按钮执行数据生成与文件写入；
2. 运行逻辑：无弹窗提示、无进度条，静默执行；
3. 异常逻辑：无论输入数值是否合理，均正常执行生成（**无参数校验**，程序不报错、不崩溃）。

## 6. CSV 文件输出规范（强制验收标准）
### 6.1 文件结构与表头
1. 首行为固定表头：`时间(ms),V1,V2,V3...Vn`
2. 列规则：
   - 第1列：时间列（单位 ms）；
   - 第2列至末尾：依次对应 `V1 ~ Vn` 通道数据，列数 = 配置通道数；
3. 数据行数：有效数据行数 = 配置的**采样点数**。

### 6.2 数值格式
1. 所有字段（时间、各通道物理量）**统一保留 2 位小数**；
2. 禁止使用科学计数法，纯十进制展示。

### 6.3 编码格式
文件编码：**UTF-8**，保证 Excel、WPS、MATLAB、Python 等软件正常读取无乱码。

### 6.4 目录与命名规则（强制）
1. 根目录：固定路径 `D:\data\`
   - 程序启动/生成文件时，若目录不存在则**自动创建**；
2. 子文件夹：每次生成自动新建文件夹，命名为**当前系统时间**，格式：`YYYYMMDD_HHMMSS`；
3. CSV 文件名：与上级文件夹同名，格式：`YYYYMMDD_HHMMSS.csv`；
4. 完整路径示例：
   ```
   D:\data\20251230_154022\20251230_154022.csv
   ```
5. 重名处理：同路径同名文件**直接覆盖**，无弹窗询问。

### 6.5 时间列规则
时间从 `0` 开始，按照「时间间隔」均匀递增，无断点、无跳变。

## 7. 非功能需求
### 7.1 性能要求
1. 启动速度：程序启动至主界面加载完成 ≤ 3 秒；
2. 生成速度：10 通道、最大采样点数范围内，数据生成+文件写入 ≤ 2 秒；
3. 内存占用：运行峰值内存 ≤ 500MB，无内存泄漏。

### 7.2 系统兼容性
- 操作系统：Windows 7 / Windows 8 / Windows 10 / Windows 11（32位、64位全兼容）；
- 文件兼容：输出 CSV 可被主流办公、数据处理软件正常解析。

### 7.3 稳定性要求
1. 连续运行 24 小时无闪退、无崩溃；
2. 非法输入（负数、超大数值、特殊字符）不会导致程序异常；
3. 磁盘权限不足时静默失败，程序不崩溃。

### 7.4 易用性要求
1. 所有配置项配有中文标签+单位说明，语义无歧义；
2. 完整流程：参数配置 →（可选加载预设）→ 点击生成，步骤极简。

## 8. 验收标准
### 8.1 功能验收
1. 全局参数可正常修改、默认值与文档一致；
2. 1~10 通道可正常切换，每通道波形选择、参数配置生效；
3. 参数变更后，波形预览实时刷新，曲线与配置逻辑一致；
4. 预设可正常保存、加载，重启程序后预设不丢失；
5. 点击【生成】可正常创建目录、输出 CSV 文件。

### 8.2 文件验收
1. 表头格式、列数量完全符合要求；
2. 时间列连续、步长准确；
3. 所有数值保留 2 位小数，编码为 UTF-8；
4. 目录结构、文件夹/文件名命名规则严格匹配文档要求。

### 8.3 非功能验收
1. 启动、生成速度、内存占用满足性能指标；
2. 在指定 Windows 系统中可正常安装、运行；
3. 异常输入、长时间运行无崩溃、闪退。
